# First Customer Playbook — v1.0.2

**Status:** First Customer Ready

## Goal

Close the first paying customer through disciplined sales execution, delivery, and onboarding.

## Daily workflow

1. **Sourcing** — add 5 agencies to `/sales/sourcing` (Germany, DACH, EU).
2. **Execution** — review `/sales/execution` (outreach, replies, meetings).
3. **Proposals** — generate PDF proposals for qualified leads.
4. **Onboarding** — start kickoff when deal is won.
5. **Portal** — customer tracks milestones in `/client-portal/onboarding`.

## Conversion path

```
Sourced lead → Discovery call → Proposal → Pilot → Won → Onboarding → Paying customer
```

## Key routes

| Route | Purpose |
|-------|---------|
| `/sales/sourcing` | Top 100 agency lists |
| `/sales/execution` | Sales execution dashboard |
| `/sales/proposals` | Proposal generator + PDF |
| `/sales/onboarding` | Customer delivery checklist |
| `/client-portal/onboarding` | Customer-facing status |
