# First Customer Metrics — v1.0.2

**Implementation:** `src/lib/sales/execution-metrics.ts`

| Metric | Description |
|--------|-------------|
| Time to close | Avg days from lead created to won |
| Pilot conversion | Won / pilot applications |
| Average deal size | Avg MRR of won deals |
| Revenue forecast | Sum of active pipeline MRR |
| Customer satisfaction | Avg portal satisfaction score |

Displayed on `/sales/execution`.

## Targets (first customer)

- Time to close: &lt; 45 days
- Pilot conversion: ≥ 25%
- Average deal size: ≥ $299 MRR
- Customer satisfaction: ≥ 80%
