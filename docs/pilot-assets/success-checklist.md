# Pilot Success Checklist — Customer-Facing

Share with pilot customers at kickoff (week 1).

---

## Week 1 — Foundation

- [ ] Complete workspace signup (or accept invite)
- [ ] Invite team members (operator + viewer roles)
- [ ] Import or create first 3 clients
- [ ] Review dashboard and client health badges

## Week 2 — Delivery

- [ ] Create first report from template
- [ ] Mark report **ready** for internal review
- [ ] Log at least 1 risk or incident

## Week 3 — Feedback

- [ ] Complete mid-pilot feedback survey
- [ ] Attend weekly review call
- [ ] Identify one automation workflow to try

## Week 4 — Portal

- [ ] Publish report to client portal
- [ ] Invite client portal user (optional)
- [ ] Configure white label basics (logo/colors) if on Business+

## Week 5 — Operations

- [ ] Run or review automation workflow
- [ ] Export compliance audit sample (CSV)
- [ ] Review usage dashboard

## Week 6 — Decision

- [ ] Complete exit questionnaire
- [ ] Decide: convert / extend / offboard
- [ ] Schedule commercial follow-up call
