## Success criteria (6 weeks)

- Weekly active usage by operations owner
- ≥ 1 published client report per month
- Feedback survey submitted (week 3 + exit)
- Documented ROI hypothesis for post-pilot conversion

---

## Conversion path

| Week | Milestone |
|------|-----------|
| 0 | Discovery call + demo |
| 1 | Onboarding + first client imported |
| 2 | First report published to portal |
| 3 | Mid-pilot feedback survey |
| 4–5 | Automation or connector connected |
| 6 | Exit questionnaire + commercial decision |
