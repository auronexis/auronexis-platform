# Pilot Playbook

## Phase 1 — Qualification (Week 0)

- Confirm ICP fit (MSP, agency, consultancy, automation firm)
- Validate 3+ clients and operations owner
- Share pilot terms, privacy, and acceptable use links

## Phase 2 — Onboarding (Week 1)

- Create workspace and invite team
- Import or create 3 demo clients
- Configure branding and portal (optional)
- Walk through dashboard, reports, and status surfaces

## Phase 3 — Value proof (Weeks 2–4)

- Publish first client report via portal
- Enable one automation or connector (if in scope)
- Review risk/incident workflow with customer

## Phase 4 — Feedback (Weeks 5–6)

- Structured feedback session
- Capture roadmap requests
- Discuss post-pilot pricing and founding customer terms
