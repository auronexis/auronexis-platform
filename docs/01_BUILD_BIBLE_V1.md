# CORE PROBLEM

Agencies build automations.

After deployment they lose visibility.

Most agencies struggle to answer:

* Which client needs attention?
* Which client is at risk?
* What value was delivered?
* Which client is profitable?
* What may fail next?

Auroranexis exists to answer those questions.

---

# GOLDEN RULE

Every feature must improve at least one of the following:

1. Visibility
2. Risk Detection
3. Client Retention
4. Operational Control
5. Profitability Awareness

If a feature does not improve one of these outcomes:

Do not build it.
